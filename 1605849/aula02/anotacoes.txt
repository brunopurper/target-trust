O processo de otimização da Base de Dados é conhecimento como Normalização e Tuning - Tunagem

Nós vamos usar o MySQL (Maria DB) e vamos utilizar o MySQL Workbench que é exclusivo para MySQL.

ENTIDADES => TABELAS
ATRIBUTOS => COLUNAS

PK => Primary Key => Chave Primaria
FK => Foreign Key => Chave Estrangeira

Tipos:

CHAR(5) => ele SEMPRE ocupa 5 "espaços" no banco
Ex.
OI      => OI_ _ _

CHAR(1) => sempre UM caractere
Ex.:
M ou F

VARCHAR(5) => ele VARIA até 5 caracteres
Ex.:
OI => vai ocupar só 2
FELIZ => vai ocupar os 5
TRISTE => vai cortar no caractere 5 (truncar)
